# Iida Kenny — Next.js Resume

A single-page resume web app built with Next.js (App Router) and styled entirely with Tailwind CSS utility classes.

## How to Run Locally
```bash
npm install
npm run dev
```
Then open http://localhost:3000 in your browser.

## Features

- [x] Next.js App Router project scaffolded with create-next-app
- [x] Resume content written as JSX in src/app/page.js
- [x] Semantic HTML elements (header, main, section, footer, h1–h3)
- [x] All styling done with Tailwind CSS utility classes via className
- [x] No custom CSS rules in globals.css
- [x] Clean, readable layout at desktop viewport
- [x] IBM certification links to Coursera credential
- [x] Projects section with links to GitHub repos
- [x] No console errors

## Known Issues / Limitations

- Layout is optimized for desktop; mobile layout is functional but not fully polished
- Project descriptions are based on repo names and may not reflect full project scope
